What to include in every website prompt

Use this structure in your prompts:

    Business type + target audience

    Visual style direction

    Colour palette

    Typography style

    Hero section style

    Navigation style

    Button style

    Image direction

    Sections required

    Trust/conversion elements

    Mobile-first behaviour

    SEO requirements

    Accessibility requirements

    What to avoid

A strong “master prompt formula”

You can reuse this for almost any niche:

text
Copy

Create a visually stunning, premium, modern website for [business type] targeting [audience].   
  
The website should feel custom-designed, elegant, high-converting, and not like a generic template. Use strong visual hierarchy, balanced spacing, modern typography, refined colour combinations, subtle shadows, tasteful borders, and high-end section layouts.  
  
Design direction:  
- Brand personality: [luxury / warm / trustworthy / clean / bold / minimal / premium]  
- Overall mood: [e.g. modern editorial, Scandinavian minimal, elegant boutique, polished corporate, inviting local business]  
- Colour palette: [list exact colours or style direction]  
- Typography: [e.g. elegant serif headings + clean sans-serif body / geometric sans-serif / refined corporate]  
- Background style: [e.g. soft gradients, textured neutrals, dark cinematic photography, subtle abstract shapes, blurred glow accents]  
- Imagery: [type of photos]  
- Buttons: visually attractive, slightly rounded, clear hover states, premium feel, excellent contrast  
- Navigation: sticky header, elegant hover states, CTA button in navbar, mobile-friendly slide-out menu  
- UI details: icons, cards, badges, trust indicators, testimonials, spacing rhythm, subtle animations on scroll  
  
Required sections:  
- Hero section with strong headline, subheadline, CTA, supporting visual  
- Services or product categories  
- Social proof / testimonials / reviews  
- About / brand story  
- Why choose us  
- FAQ  
- Final CTA  
- Footer with SEO-friendly internal links and contact details  
  
SEO requirements:  
- Semantic HTML structure  
- Clear H1, H2, H3 hierarchy  
- Local SEO-ready content where relevant  
- Optimised section headings  
- Strong conversion-focused copy prompts  
- Fast-loading design, lightweight effects, mobile-first layout  
  
UX requirements:  
- Excellent readability  
- Clear CTA placement  
- High contrast  
- Accessible forms  
- Easy scanning  
- Strong mobile responsiveness  
- Professional spacing and alignment  
  
Avoid:  
- Generic stock-template appearance  
- Random colours  
- Overcrowded layouts  
- Excessive animations  
- Cheap-looking gradients  
- Tiny text  
- Unclear CTA buttons  
- Cluttered navigation  
  
Make the site feel premium, polished, and worthy of a top design agency portfolio.  

Pro tip: use a 2-step prompt method

Instead of asking for the full website in one go, do this:
Step 1: ask for the design system first

text
Copy
Before building the website, define the visual direction first:  
- brand style  
- colour palette with hex codes  
- typography pairing  
- button styles  
- card styles  
- background treatment  
- image style direction  
- section spacing system  
- animation style  
- desktop and mobile layout principles  
  
Give me 3 distinct creative directions, all premium and conversion-focused, then implement the best one.  

Step 2: ask for a polish pass

text
Copy
Now refine the design to look more premium and less template-like.  
Improve:  
- hero composition  
- typography scale  
- button styling  
- navbar elegance  
- section transitions  
- background depth  
- icon consistency  
- card spacing  
- mobile responsiveness  
- premium micro-interactions  
  
Make it feel like a professionally art-directed website, not an AI-generated draft.

Best follow-up prompts after first draft

Once the first version is generated, these follow-up prompts usually improve it a lot:
To improve visuals

text
Copy
Refine the visual design so it feels more premium, polished, and memorable. Improve typography hierarchy, colour balance, spacing, card styling, button aesthetics, navbar elegance, and background depth.  

To improve the hero section

text
Copy
Redesign the hero section to be more visually striking and emotionally engaging. Improve headline placement, image composition, CTA prominence, overlay treatment, and the premium feel of the first fold.  

To improve buttons and navigation

text
Copy
Make the buttons and navigation feel more elegant and modern. Add refined hover states, better spacing, stronger contrast, and a more premium interaction style.  

To reduce template feel

text
Copy
Make the layout less boxy and less template-like. Introduce better section flow, varied composition, subtle asymmetry where appropriate, stronger visual rhythm, and more custom detailing.
